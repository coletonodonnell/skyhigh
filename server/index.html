Sky High

The SkyHigh project is the final project submission for COP3530. Group members of this project are: Coleton O'Donnell, Maria Nicos Alain Pasaylo, and Akash Boggavarapu. This project utilizes flight data from the year 2018. Given airport code origin and destination, the average ticket prices of various airline companies will be displayed along with a map visual. The project utilizes a BTree Set and a Hash Set to sort and store the flight data. A total of 500,000 random flights were taken from the flight dataset source.
